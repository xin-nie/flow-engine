# flow-engine

A simple flow engine that evaluates rules against passing object.

## Install

Install yarn first:

```
npm install -g yarn
```

And install dependencies:

```
yarn install
```

## Usage

Build app first:

```
yarn run build
```

Then you can start playing:

```
yarn run serve
```

Or you can start dev mode with file watcher:

```
yarn start
```

## Test

```
yarn run test
```
